# TechPulse

A clean, developer-focused dashboard to track latest tech updates from multiple sources in one place.

## Features

- Unified tech news feed
- Topic-based filtering
- Region filtering (Global / India / Malayalam)
- Modern UI (dark, minimal)
- Fast and lightweight

## Tech Stack

- React (Vite)
- Tailwind CSS
- Node.js (Fastify)
- RSS Aggregation
